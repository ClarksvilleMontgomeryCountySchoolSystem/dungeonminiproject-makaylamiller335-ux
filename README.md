# Dungeon Escape

You are trapped in a dungeon. Complete all five tasks to escape — or meet your `Doom:`

## Requirements
- Complete all five tasks
- The `not` operator must be used exactly twice across all tasks
- Each use of `not` must be part of a conditional statement

---

## Task 1 — The Torch

You wake up on a cold stone floor. Your torch may be lit — and that changes everything.

1. Create a boolean variable `torch_lit`
2. Write a conditional statement with an else branch
3. Inside each branch, define `outcome` as a string beginning with the correct prefix followed by your own creative text
   - Use `Flicker:` for the success branch
   - Use `Doom:` for the other branch
4. Print `outcome`
